# FRC Program Control Layer

This directory is the human-facing control layer for the FRC research program.
The machine-readable counterpart is `../../registry/papers.json`.

Read in this order:

1. `AUTHORITATIVE_LOCATIONS.md` — what is canonical and what is an export,
   build product, working copy, or historical evidence.
2. `ROADMAP.md` — research sequence and promotion gates.
3. `CHANGELOG.md` — substantive program and publication changes.
4. `PUBLICATION_WORKFLOW.md` — required release and Zenodo procedure.
5. `FILE_CLASSIFICATION.md` — how scattered material is retained safely.

This layer is intentionally additive. It does not erase drafts, agent handoffs,
or prior publications.
